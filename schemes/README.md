Electromechanical diagrams
====

This directory contains a [connection diagram](https://github.com/csvprobotica/RoSGhost/blob/main/schemes/Diagrama%20de%20Conexi%C3%B3n.png) where all the ports used are distributed, a process diagram which details all the actions to be performed and a list of electronic components in which each of the elements used are described.
